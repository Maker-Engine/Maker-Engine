Maker Engine Roadmap

This roadmap outlines the planned development goals for Maker Engine.
Features and milestones may change as development progresses.

---

Phase 1 — Foundation

Goal: Build the core systems of the engine.

Core

- Engine core loop
- Scene management
- GameObject / Entity system
- Event system

Rendering

- Basic sprite rendering
- Camera system
- Layer system

Input

- Keyboard input
- Mouse input
- Basic touch input

Audio

- Sound playback
- Music playback
- Volume control

---

Phase 2 — Scripting

Goal: Make the engine flexible through scripting.

Lua Integration

- Lua runtime integration
- Script loading system
- Script events

Script Events

Examples:

- "onCreate()"
- "onUpdate()"
- "onBeat()"
- "onDestroy()"

Mod Support

- Mods folder
- Automatic script loading
- Asset override system

---

Phase 3 — Gameplay Systems

Goal: Add systems commonly needed for games.

Systems

- Animation system
- Collision detection
- Physics helpers
- Timer system
- Tween system

UI

- Text rendering
- Buttons
- Menus
- HUD system

---

Phase 4 — Tools

Goal: Improve development workflow.

Tools

- Debug console
- Asset loader
- Logging system

Editor (Optional)

Future visual tools may include:

- Scene editor
- Sprite animation editor
- UI editor

---

Phase 5 — Platform Support

Goal: Expand platform compatibility.

Supported platforms:

- Windows
- Linux
- macOS
- Android
- HTML5

Possible future platforms:

- iOS
- WebAssembly

---

Phase 6 — Advanced Features

Goal: Add powerful engine capabilities.

Advanced Systems

- Save system
- Networking support
- Shader support
- Particle system

Optimization

- Performance improvements
- Memory optimization
- Asset streaming

---

Future Ideas

Possible features that may be added later:

- Visual scripting
- Plugin system
- Built-in level editor
- Asset pipeline tools
- Marketplace for mods

---

Long-Term Vision

Maker Engine aims to become:

- A lightweight 2D game engine
- Highly moddable
- Friendly for beginners and advanced developers
- Capable of exporting to multiple platforms

---

⭐ Development is ongoing.
Contributions and feedback are welcome!
